# PostgreSQL - Estudos e Exercícios

Repositório criado para armazenar meus estudos de PostgreSQL, contendo scripts, exercícios resolvidos e exemplos práticos desenvolvidos durante meu aprendizado de Banco de Dados.

## 📚 Conteúdos estudados

* Criação de tabelas (`CREATE TABLE`)
* Inserção de dados (`INSERT`)
* Alteração de registros (`UPDATE`)
* Exclusão de registros (`DELETE`)
* Consultas com `SELECT`
* Filtros com `WHERE`
* Ordenação com `ORDER BY`
* Operadores lógicos (`AND`, `OR`)
* Pesquisa com `LIKE`
* Tratamento de valores `NULL`
* Funções agregadas:

  * `COUNT`
  * `SUM`
  * `AVG`
  * `MIN`
  * `MAX`
* Agrupamentos com `GROUP BY`
* Filtros de grupos com `HAVING`
* Chaves Primárias (PK)
* Chaves Estrangeiras (FK)
* Relacionamentos entre tabelas
* JOINs:

  * `INNER JOIN`
  * `LEFT JOIN`

## 📂 Estrutura

postgresql/

## 🎯 Objetivo

Praticar conceitos fundamentais de Banco de Dados Relacional utilizando PostgreSQL, desenvolvendo uma base sólida para aplicações Java, JDBC e Spring Boot.
